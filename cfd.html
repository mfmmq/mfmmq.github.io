CFD-Assisted Study of Diesel Emission Generation
Engines and Combustion Group, University of Oxford, 2017-2018

For my fourth year MSci project, I analysed test engine data and simulated engine data. This involed:

  • Running test simulations in CONVERGE, a proprietary CFD software
  • Validating simulation results with real engine data
  • Using visual interpretations and scripts to draw conclusions combining data analytics and fundamental thermofluid theory

    • Successful completion of this project also included challenges such as:
  • How to validate data
  • How to accurately represent the data
  • 3D and time
